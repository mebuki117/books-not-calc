## 説明
1.8で交易に必要な本の数を求めるツール。  
計算は用いておらず、ルックアップテーブルを参照する。


## 動作環境
[Python](https://www.python.org/)をインストール。  
バージョンは新しければ何でもいいと思うが、開発環境はv3.12.10。

## 使い方
1. `books not calc.pyw`を起動
2. `emr cost`、`eye cost`、`need eyes`を順に入力
3. 必要な`本の数`と、`本のスタック数+余り`が表示される
   
## 備考
- `emr cost`：1エメラルドに必要な本の数
- `eye cost`：1アイに必要なエメラルドの数
- `need eyes`：必要なアイの数
